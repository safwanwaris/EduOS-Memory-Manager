 EduOS Memory Manager Simulation

📌 Project Overview

This project simulates memory management in a lightweight operating system (EduOS) designed for smart classrooms.

It demonstrates how different page replacement algorithms perform under limited memory conditions.


⚙️ Features

* Paging simulation
* FIFO Page Replacement
* LRU Page Replacement
* Optimal Page Replacement
* Step-by-step frame visualization
* Page fault calculation


🧠 Algorithms Implemented

* First-In First-Out (FIFO)
* Least Recently Used (LRU)
* Optimal Algorithm

▶️ How to Run

```bash
python memory_manager.py
```

📊 Sample Input

Reference String:
[7, 0, 1, 2, 0, 3, 0, 4]

Frames: 3


📈 Output

Displays:

* Frame allocation step-by-step
* Page hits and faults
* Total faults for each algorithm

🎯 Conclusion

This project highlights how efficient memory management improves system performance, especially in low-resource environments like smart classrooms.
